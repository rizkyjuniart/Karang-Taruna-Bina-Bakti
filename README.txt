WEB STREAMLIT KARANG TARUNA BINA BAKTI

Cara menjalankan di VS Code:

1. Ekstrak file ZIP.
2. Buka folder hasil ekstrak di VS Code.
3. Buka terminal VS Code.
4. Install Streamlit:
   pip install streamlit

5. Jalankan web:
   streamlit run app.py

Jika ingin upload online ke Streamlit Cloud:
1. Upload app.py dan requirements.txt ke GitHub.
2. Buka https://share.streamlit.io
3. Pilih repository GitHub.
4. Main file path isi:
   app.py
5. Klik Deploy.

Bagian yang bisa diganti:
- Nama pengurus
- Nomor WhatsApp
- Email
- Alamat
- Jumlah anggota/program/kegiatan
- Jadwal kegiatan
